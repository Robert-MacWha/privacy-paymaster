# Privacy Paymasters

This repo contains implementations of 4337 paymasters for various privacy protocols (Tornado Cash, Railgun). These paymasters are designed
to enable private unshields to new user accounts without needing a trusted per-protocol relayer. Users pay for their transactions using the private assets they hold or are unshielding.

For more information, see the docs in `docs/`. For information on specific protocol implementations, see the relevant subdirectories and docstrings in `contracts/accounts/`.
